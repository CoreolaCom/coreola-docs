# `phrase.ts`

Translation lookup with dev-mode hints for missing keys.

- `phrase(phrases, translation, options, target)` — Returns a lookup function that translates `phrases[key]` against the active i18n instance, with console hints in dev mode for missing keys.

Powers the `usePhrase` hook — see Hooks for the React-side API.

- Source: `src/helpers/phrase.ts`
- Import: `import { phrase } from 'helpers/phrase'`

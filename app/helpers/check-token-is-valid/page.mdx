# `checkTokenIsValid.ts`

JWT access-token expiration check.

- `checkTokenIsValid(accessToken, now): boolean` — Decodes the token's `exp` claim and returns `true` if it is still in the future.

Used by the auth slice refresh logic and the ability/flag setup to decide whether to fetch protected data.

- Source: `src/helpers/checkTokenIsValid.ts`
- Import: `import { checkTokenIsValid } from 'helpers/checkTokenIsValid'`

# `dirtyData.ts`

Pick the modified fields out of a react-hook-form state.

- `dirtyData(dirtyFields, allValues)` — Returns only the dirty fields from `react-hook-form` and trims any string values along the way. The shape ready to send to a PATCH endpoint.

- Source: `src/helpers/dirtyData.ts`
- Import: `import { dirtyData } from 'helpers/dirtyData'`

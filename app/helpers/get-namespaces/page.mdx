# `getNamespaces.ts`

Collect all i18n namespaces declared in a route tree.

- `getNamespaces(routes)` — Walks a route tree and returns all `namespace` declarations in traversal order.

Used at app bootstrap to know which translation bundles to load.

- Source: `src/helpers/getNamespaces.ts`
- Import: `import { getNamespaces } from 'helpers/getNamespaces'`

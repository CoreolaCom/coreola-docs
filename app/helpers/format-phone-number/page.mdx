# `formatPhoneNumber.ts`

Format a phone number using the project mask configuration.

- `formatPhoneNumber(phoneNumber, countryCode?)` — Formats a raw phone string with auto-detected country when `countryCode` is omitted. Returns `''` for non-string/non-number input.

- Source: `src/helpers/formatPhoneNumber.ts`
- Import: `import { formatPhoneNumber } from 'helpers/formatPhoneNumber'`

# `persistPurgeCheck.ts`

Auto-purge persisted state on app version bump.

- `persistPurgeCheck()` — Compares the stored app version with `import.meta.env.VITE_APP_VERSION`. On a bump, clears persisted slices, busts the service worker cache, and force-reloads the page.

Call once at app startup.

- Source: `src/helpers/persistPurgeCheck.ts`
- Import: `import { persistPurgeCheck } from 'helpers/persistPurgeCheck'`

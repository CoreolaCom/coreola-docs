# `usePhrase`

Returns a scoped `t` function keyed under a `target` prefix.

```ts
usePhrase(target: string, namespace?: string, options?: UseTranslationOptions): TFunction
```

**Returns** — A `TFunction` that falls back to the phrase key when a translation is missing.

**Use when** — When a feature has many short labels under a shared prefix — e.g. `usePhrase('form')`.

- Source: `src/hooks/usePhrase.ts`
- Import: `import usePhrase from 'hooks/usePhrase'`

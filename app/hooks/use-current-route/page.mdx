# `useCurrentRoute`

Resolves the currently matched application route with `pathnameBase` normalized (route params stripped).

```ts
useCurrentRoute(): MatchType | undefined
```

**Returns** — The current route match, or `undefined` when no route matches.

**Use when** — When you need a stable identifier for the active route — breadcrumbs, scoped UI, analytics.

- Source: `src/hooks/useCurrentRoute.ts`
- Import: `import useCurrentRoute from 'hooks/useCurrentRoute'`

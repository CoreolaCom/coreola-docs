# `useIsInViewport`

Tracks viewport intersection via `IntersectionObserver`.

```ts
useIsInViewport(ref: RefObject<HTMLElement | null>): boolean
```

**Returns** — `true` while the element is visible, otherwise `false`.

**Use when** — For lazy-load triggers, infinite-scroll sentinels, and entrance animations.

- Source: `src/hooks/useIsInViewport.ts`
- Import: `import useIsInViewport from 'hooks/useIsInViewport'`

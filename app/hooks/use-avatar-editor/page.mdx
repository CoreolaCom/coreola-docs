# `useAvatarEditor`

End-to-end state for an avatar upload + crop + rotate + zoom flow.

```ts
useAvatarEditor(params: UseAvatarEditorParams): UseAvatarEditorResponse
```

**Returns** — Editor state, dropzone props, crop callbacks, and handlers (clear / cancel / toggle / reset).

**Use when** — When building a profile photo or avatar field with image cropping.

- Source: `src/hooks/useAvatarEditor.ts`
- Import: `import useAvatarEditor from 'hooks/useAvatarEditor'`

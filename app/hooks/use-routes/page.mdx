# `useRoutes`

Returns the application route tree available to the current user — filtered by CASL ability and feature flags, with `t()` applied to titles.

```ts
useRoutes(): Route[]
```

**Returns** — A `Route[]` array.

**Use when** — For navigation, menus, or any UI that should only show authorized routes.

- Source: `src/hooks/useRoutes.ts`
- Import: `import useRoutes from 'hooks/useRoutes'`

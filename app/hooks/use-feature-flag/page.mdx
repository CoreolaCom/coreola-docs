# `useFeatureFlag`

Loads feature flags for the authenticated user and exposes a `canFeatureFlag(key)` check plus a `featureFlagsLoaded` indicator.

```ts
useFeatureFlag(): { featureFlags; canFeatureFlag; featureFlagsLoaded }
```

**Returns** — `{ featureFlags, canFeatureFlag, featureFlagsLoaded }`.

**Use when** — When conditionally rendering features gated by server-controlled flags.

- Source: `src/hooks/useFeatureFlag.ts`
- Import: `import useFeatureFlag from 'hooks/useFeatureFlag'`

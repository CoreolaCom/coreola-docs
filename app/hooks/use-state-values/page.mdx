# `useStateValues`

Manages a typed object of related state values as one unit, with an overloaded setter (single key or partial object).

```ts
useStateValues<T>(defaultValues: T): { values: T; setStateValue: SetStateValue<T>; resetValues: VoidFunction }
```

**Returns** — `{ values, setStateValue, resetValues }`.

**Use when** — When you'd otherwise call `useState` five times for related fields.

- Source: `src/hooks/useStateValues.ts`
- Import: `import useStateValues from 'hooks/useStateValues'`

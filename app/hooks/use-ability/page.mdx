# `useAbility`

Builds the current user's CASL ability object from the ability matrix stored in the auth slice.

```ts
useAbility(): AppAbility
```

**Returns** — A memoized `AppAbility` instance.

**Use when** — When checking user permissions in RBAC-protected UI; pair with `<Can>` or call `ability.can(action, subject)`.

- Source: `src/hooks/useAbility.ts`
- Import: `import useAbility from 'hooks/useAbility'`

# `useWatchMemoize`

Returns the same value reference across renders as long as the value remains deeply equal (lodash `isEqual`).

```ts
useWatchMemoize<T>(value: T): T
```

**Returns** — A stable value reference.

**Use when** — As a stability shim before passing complex values into `useEffect` / `useMemo` dependency arrays.

- Source: `src/hooks/useWatchMemoize.ts`
- Import: `import useWatchMemoize from 'hooks/useWatchMemoize'`

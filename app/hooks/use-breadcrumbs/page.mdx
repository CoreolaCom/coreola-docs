# `useBreadcrumbs`

Builds the breadcrumb chain for the current route by combining matched routes with Redux-stored breadcrumb section overrides.

```ts
useBreadcrumbs(): { branch: CrumbsSection[]; lastItem?: CrumbsSection }
```

**Returns** — `{ branch: CrumbsSection[]; lastItem?: CrumbsSection }`.

**Use when** — Inside your breadcrumb component to render the trail; pair with `useBreadcrumbSectionTitle` to override.

- Source: `src/hooks/useBreadcrumbs.ts`
- Import: `import useBreadcrumbs from 'hooks/useBreadcrumbs'`

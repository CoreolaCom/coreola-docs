# `useRouteValuesGuard`

Validates that the URL params on each matched route are listed in that route's `values` configuration.

```ts
useRouteValuesGuard(routeMatches: RouteMatch[]): boolean
```

**Returns** — `true` when every match is valid, otherwise `false`.

**Use when** — To short-circuit rendering when a URL param has an invalid enum-like value, before the page fetches data.

- Source: `src/hooks/useRouteValuesGuard.ts`
- Import: `import useRouteValuesGuard from 'hooks/useRouteValuesGuard'`

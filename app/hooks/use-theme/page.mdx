# `useTheme`

Builds and memoizes the active MUI `Theme` based on the alias stored in the `app` slice.

```ts
useTheme(): Theme
```

**Returns** — A MUI `Theme`.

**Use when** — In the app root to compute the theme passed to `<ThemeProvider>`. Prefer MUI's own `useTheme()` inside components.

- Source: `src/hooks/useTheme.ts`
- Import: `import useTheme from 'hooks/useTheme'`

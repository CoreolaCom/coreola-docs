# `useSiblingsRoutes`

Builds sibling routes and tab options from the current route's parent.

```ts
useSiblingsRoutes(): { siblingsRoutes: Route[]; siblingsTabs: TabOption[] }
```

**Returns** — `{ siblingsRoutes, siblingsTabs }`.

**Use when** — When rendering tabbed sub-navigation between siblings of the current route.

- Source: `src/hooks/useSiblingsRoutes.ts`
- Import: `import useSiblingsRoutes from 'hooks/useSiblingsRoutes'`

# `useBreadcrumbSectionTitle`

Side-effectful: dispatches Redux actions to set a temporary breadcrumb section title for the current route and clears it on unmount.

```ts
useBreadcrumbSectionTitle(props: UseBreadcrumbSectionTitleProps): void
```

**Returns** — Nothing.

**Use when** — On detail pages that need to inject a dynamic name (entity title) into the breadcrumb chain.

- Source: `src/hooks/useBreadcrumbSectionTitle.ts`
- Import: `import useBreadcrumbSectionTitle from 'hooks/useBreadcrumbSectionTitle'`

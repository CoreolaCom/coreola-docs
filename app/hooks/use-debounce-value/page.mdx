# `useDebounceValue`

Returns a debounced copy of `value` that only updates after `timeout` ms of no further changes.

```ts
useDebounceValue<T>(value: T, timeout: number, defaultValue?: T): T | undefined
```

**Returns** — The debounced value, or `defaultValue` before the first debounce.

**Use when** — To throttle search inputs, window resize handlers, or any value driving an expensive effect.

- Source: `src/hooks/useDebounceValue.ts`
- Import: `import useDebounceValue from 'hooks/useDebounceValue'`

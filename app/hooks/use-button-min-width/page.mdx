# `useButtonMinWidth`

Returns the standard minimum width for action buttons based on the current MUI breakpoint.

```ts
useButtonMinWidth(): number
```

**Returns** — `150` on `md+`, otherwise `100`.

**Use when** — On buttons in dialog footers and forms to keep widths consistent across the app.

- Source: `src/hooks/useButtonMinWidth.ts`
- Import: `import useButtonMinWidth from 'hooks/useButtonMinWidth'`
